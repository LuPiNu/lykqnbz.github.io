> “Sass 是一门高于 CSS 的元语言，它能用来清晰地、结构化地描述文件样式，有着比普通 CSS 更加强大的功能。Sass 能够提供更简洁、更优雅的语法，同时提供多种功能来创建可维护和管理的样式表。”

## 关于安装
在Ruby Command环境下：
- 安装 Sass：`gem install sass`
- 查看 Sass：`sass -v`
- 更新 Sass：`gem update sass`
- 卸载 Sass：`gem uninstall sass`
